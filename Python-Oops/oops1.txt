------------ obeject oriented programming ----------------------------

it is advanced programming technique that allows you to
create a modular program that is easier to understand and maintain.
it is a way of programming that is based on the concept of objects,
which can contain data in the form of fields (often known as attributes or properties),
and code, in the form of procedures (often known as methods).


-------------------- importants of OOP ----------------------------
it is very important concept in programming.
when we develop a software we Follow principal of SDC(Software Development Cycle)
1. Requirement Analysis
2. Design
3. Coding
4. Testing
5. Deployment
6. Maintenance

in design phase we use OOP concept to design our software.(Best Approch/Practics)
it is very useful in large software development.
oops used each type of software development (web, mobile, desktop, ai app ,ML APP etc)
